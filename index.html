Mini-project 2: Web Architecture

Khám Phá Hạ Tầng Mạng & Cấu Trúc URL

Hệ thống trực quan hóa giúp phân tích chuyên sâu về cơ chế hoạt động của Tên miền (Domain), Không gian lưu trữ (Hosting) và định vị tài nguyên Internet.

Bắt Đầu Trực Quan Hóa URL ➔

1. Ba Trụ Cột Định Hình Một Website

Những thành phần cốt lõi bắt buộc phải có để một ứng dụng web vận hành trên Internet.

🏠

Domain Name (Tên miền)

Là chuỗi ký tự chữ viết dễ nhớ dùng để định danh vị trí của một website, thay thế cho các địa chỉ IP số (Internet Protocol) khô khan của máy chủ.

  • Top-Level Domain (TLD): .com, .vn, .org
  • Second-Level Domain (SLD): google, github
  • Chức năng: Đại diện cho thương hiệu số của doanh nghiệp.
📦

Web Hosting (Lưu trữ)

Là không gian đĩa cứng được thuê trên một máy chủ (Server) cấu hình mạnh, luôn kết nối mạng 24/7. Đây là nơi chứa toàn bộ mã nguồn, hình ảnh và dữ liệu của trang web.

  • Thành phần chứa: Source code (HTML, CSS, JS), Database.
  • Tiêu chuẩn kĩ thuật: Băng thông rộng, Uptime 99.9%, mã hóa bảo mật.
  • Nền tảng mẫu: GitHub Pages (Dịch vụ hosting tĩnh chuyên dụng).
🔗

URL (Đường dẫn đồng nhất)

Viết tắt của Uniform Resource Locator. Đây là chuỗi định vị tài nguyên đồng nhất, chỉ ra chính xác phương thức truy cập và vị trí tệp tin trên mạng toàn cầu.

  • Phạm vi trỏ: Trang HTML, hình ảnh, tài liệu PDF, API endpoint.
  • Cấu trúc chuẩn: Giao thức + Tên miền + Thư mục + Tệp tin.
  • Đặc tính: Độc nhất, không trùng lặp trên mạng Internet toàn cầu.

2. Bảng Phân Tích Thực Thể Hạ Tầng Web

Hệ thống hóa cách thức các thành phần mạng máy tính phối hợp xử lý gói tin yêu cầu (Request).

Thành Phần Hệ Thống Bản Chất Kỹ Thuật Ví Dụ Thực Tế Vai Trò Khi Người Dùng Truy Cập
Giao thức (Protocol) Tập hợp quy chuẩn mã hóa để trao đổi thông tin dữ liệu an toàn. HTTP, HTTPS, FTP Trình duyệt đọc để thiết lập kênh kết nối mã hóa bảo mật dữ liệu SSL/TLS.
Hệ thống DNS Hệ thống phân giải tên miền (Giống như danh bạ điện thoại Internet). DNS Server toàn cầu: 8.8.8.8 Dịch tên miền dạng chữ (như `github.com`) sang địa chỉ số IP (`140.82.113.3`).
Web Server Máy chủ vật lý chạy phần mềm quản trị và phân phối mã nguồn. Nginx, Apache, GitHub Pages Server Tiếp nhận Request từ Client, tìm kiếm tệp tin tương ứng và trả mã phản hồi (Response).

3. Bộ Trực Quan Hóa Cấu Trúc URL Tương Tác

Nhấp chuột trực tiếp vào từng phân đoạn màu sắc của chuỗi URL bên dưới để xem định nghĩa chức năng tương ứng.

Mẫu URL 01: Đường dẫn chuỗi truy vấn động (Dynamic URL Thương mại điện tử)

https://shopee.vn/search?keyword=laptop

💡 Vui lòng bấm vào một thành phần màu sắc của chuỗi URL phía trên để xem phân tích...

🛡️ Giao thức (Protocol) - https://

Hypertext Transfer Protocol Secure. Giao thức truyền tải siêu văn bản bảo mật, đảm bảo toàn bộ thông tin trao đổi giữa trình duyệt và Shopee được mã hóa an toàn, chống nghe lén gói tin.

🌐 Tên miền chính (Host / Domain) - shopee.vn

Địa chỉ định danh thương hiệu của sàn thương mại điện tử Shopee tại Việt Nam, trỏ về cụm máy chủ xử lý dữ liệu của Shopee thông qua hệ thống phân giải DNS.

📂 Đường dẫn (Path) - /search

Chỉ định đường dẫn thư mục logic hoặc router con trên máy chủ chịu trách nhiệm gọi module thực hiện chức năng tìm kiếm sản phẩm.

⚙️ Chuỗi truy vấn (Query String) - ?keyword=laptop

Thành phần truyền dữ liệu biến động lên Server. Bắt đầu bằng dấu ? và chứa cặp Tham-số=Giá-trị. Ở đây tham số keyword có giá trị dữ liệu là laptop.

Mẫu URL 02: Đường dẫn tệp tin tĩnh trên Hosting GitHub Pages (Static URL)

https://trungnghia.github.io/mini-project-2/index.html

💡 Vui lòng bấm vào một thành phần màu sắc của chuỗi URL phía trên để xem phân tích...

🛡️ Giao thức an toàn (Protocol) - https://

Giao thức bảo mật tiêu chuẩn giúp trang web cá nhân của bạn tránh được cảnh báo nguy hiểm từ trình duyệt Chrome hoặc Cốc Cốc.

🏢 Tên miền phụ (Subdomain) - trungnghia.

Một nhánh con tách lập từ tên miền gốc. GitHub sử dụng chính Tên tài khoản (Username) của bạn để định danh phân vùng lưu trữ cá nhân.

🌐 Tên miền gốc dịch vụ (Root Domain) - github.io

Tên miền hệ thống máy chủ dịch vụ cung cấp giải pháp deploy hosting tĩnh miễn phí cấp phát bởi tập đoàn GitHub.
