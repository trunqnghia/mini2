# Mini-project 2: URL Explorer Tĩnh

Hệ thống cung cấp kiến thức hạ tầng web cốt lõi được xây dựng chuẩn cấu trúc Front-end phân tách rõ ràng.

## 🚀 Công nghệ ứng dụng
- **HTML5 Semantic**: Sử dụng cấu trúc thẻ ngữ nghĩa điều hướng chuẩn hóa dữ liệu (`<nav>`, `<article>`, `<main>`).
- **CSS3 Grid & Variables**: Quản lý layout dạng lưới responsive tự động co giãn, tối ưu hóa tái sử dụng mã thông qua biến `:root`.
- **Vanilla JavaScript**: Quản trị logic sự kiện lắng nghe thao tác chuột để tương tác hiển thị dữ liệu động (Click-to-explain).

## 📁 Sắp xếp cấu trúc Repository
- `/css/style.css` - Tách biệt lớp thiết kế giao diện.
- `/js/script.js` - Quản lý chức năng nghiệp vụ tương tác hệ thống.
- `index.html` - Bộ khung lưu trữ dữ liệu chính.
